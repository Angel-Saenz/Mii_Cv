# Resumen del Proyecto: Portafolio Inmersivo 3D - Ing Angel Saenz

Este proyecto es un portafolio profesional de alto impacto visual desarrollado con un stack moderno enfocado en la interactividad y el rendimiento.

## 🚀 Tecnologías Principales
- **Framework:** React 19 (TypeScript)
- **Renderizado 3D:** Three.js con @react-three/fiber y @react-three/drei
- **Animaciones:** Framer Motion y Lucide React
- **Estilos:** Vanilla CSS con un enfoque minimalista y responsivo
- **Herramienta de Construcción:** Vite 6

## ✨ Características Destacadas
- **Ambiente Dinámico:** Cambia entre un entorno de "Día" (nubes y aves 3D) y "Noche" (estrellas y atmósfera oscura) mediante un interruptor de tema.
- **Centro 3D Interactivo:** Una esfera central animada con distorsión de malla en tiempo real que reacciona a la navegación.
- **Onda de Proximidad:** El nombre principal reacciona al ratón con un efecto de ola donde las letras adyacentes se elevan suavemente.
- **Secciones Adaptativas:** Animaciones de entrada (spring physics) que se activan al hacer scroll, optimizadas para dispositivos móviles.
- **Despliegue Automatizado:** Configuración de GitHub Actions para despliegue continuo (CI/CD) a GitHub Pages.

## 🛠️ Configuración Técnica
- **Base Path:** Configurado dinámicamente para soportar tanto desarrollo local como producción en subcarpetas de GitHub.
- **Compatibilidad:** Ajustes específicos para asegurar el funcionamiento de Three.js en el nuevo motor de React 19.
